# Психогеография Метасистемы

Next.js 14 прототип астрологического интерфейса. Включает карты транзитов, архетипический слой, рекомендации и профиль пользователя.

## Скрипты
- `npm install` — установить зависимости.
- `npm run dev` — запустить локальную разработку.
- `npm run build` — production-сборка.

Проект использует TailwindCSS, Framer Motion и d3-force для анимаций и визуализаций.
